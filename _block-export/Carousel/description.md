# Carousel Block

A feature-rich, accessible carousel/slideshow component for AEM EDS.

## Features

- **Layout variations:** 4 layout types
  - Full-grid (expandable)
  - Full-width (edge-to-edge)
  - Image-only-medium (horizontal scroll)
  - Image-only-large (horizontal scroll)

- **Media support:** Image and Brightcove video with custom play/pause controls

- **Auto-play:** 5-second intervals, pauses on user interaction

- **Navigation:**
  - Touch swipe on mobile
  - Keyboard accessible (Arrow keys, Home, End)
  - ARIA compliant with live regions for slide announcements

- **Responsive:** Mobile-first with breakpoints at 800px and 1080px

- **Additional capabilities:**
  - CTA buttons with links
  - Scroll-triggered expand/collapse animation for full-grid layout
  - Lazy loading of video players for performance
  - Analytics tracking integration
  - Universal Editor authoring support
  - Fade transitions (full-grid/full-width) or horizontal scroll (image-only layouts)
  - Bottom margin and visibility toggle support when used inside grid-columns
